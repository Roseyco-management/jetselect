## Technical Implementation

### HTML Pattern (Pillar Pages)

```html
<!-- Hero section -->
<section class="hero" style="background-image: url('../images/jets/luxury-jet-interior-cabin.jpg');">
  <!-- Content -->
</section>

<!-- Section backgrounds -->
<section class="content-section with-image">
  <div class="section-image">
    <img src="../images/jets/private-jet-runway-front.jpg" alt="Private jet ready for takeoff" loading="lazy">
  </div>
  <!-- Content -->
</section>
```

### CSS Considerations

```css
/* Hero backgrounds - maintain existing pattern */
.hero {
  background-size: cover;
  background-position: center;
  position: relative;
}

.hero::before {
  /* Teal tint overlay (from Phase 3) */
  background: rgba(17, 90, 78, 0.4);
}

/* Lazy loading for section images */
img[loading="lazy"] {
  opacity: 0;
  transition: opacity 0.4s ease;
}

img[loading="lazy"].loaded {
  opacity: 1;
}
```

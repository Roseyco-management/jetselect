<implementation_methodology>

## Step-by-Step Process for Each Page

### 1. Read Page Content
- Open HTML file, read actual content (not just URL)
- Identify main topic, subtopics, key concepts
- Note natural linking opportunities (where reader would benefit from more info)

### 2. Identify Link Opportunities
- Where does content mention related topics?
- Where would a reader want to learn more?
- Where are concepts that have dedicated pages?
- Where does content reference processes, costs, locations, aircraft?

### 3. Select Target Pages
- Use Phase 8 architecture to identify correct parent pillar
- Identify related cluster pages (same topic area)
- Identify relevant supporting pages

### 4. Craft Anchor Text
- Use natural language from surrounding context
- Include target keyword when natural (not forced)
- Vary anchor text (don't repeat same phrase)
- Action-oriented when appropriate ("ontdek", "lees", "bekijk")

### 5. Place Links Strategically
- Early in content (first 2-3 paragraphs = high value)
- In context where reader would naturally click
- Not in same sentence/paragraph (spread out)
- Not in intro/conclusion only (throughout body)

### 6. Verify Link Density
- Count total words
- Count links added
- Ensure 2-5% density (e.g., 2,000 words = 4-10 links optimal)

### 7. Update HTML
- Use Edit tool to add `<a href="/path/">anchor text</a>`
- Preserve existing HTML structure
- Don't break paragraphs or formatting

</implementation_methodology>

## Editorial Linking Patterns (Examples)

### Pattern 1: Concept Introduction → Deep Dive Link
```html
<p>De kosten van privéjet-eigendom gaan veel verder dan alleen de aankoopprijs.
Naast de initiële investering moet u rekening houden met
<a href="/gids/kosten/jaarlijks-budget/">jaarlijkse operationele kosten</a>
die variëren van €200.000 tot meer dan €1,5 miljoen per jaar, afhankelijk
van het vliegtuigtype en gebruiksfrequentie.</p>
```

### Pattern 2: Decision Point → Comparison Link
```html
<p>Voordat u besluit een privéjet te kopen, is het cruciaal om te evalueren
of eigendom economisch zinvol is voor uw gebruikspatroon. Als u minder dan
100 vlieguren per jaar vliegt, kan
<a href="/gids/huren-vs-kopen/">charteren een kosteneffectiever alternatief</a>
zijn dan volledig eigendom.</p>
```

### Pattern 3: Process Step → Detailed Guide Link
```html
<p>Na het selecteren van uw gewenste vliegtuig is de volgende cruciale stap
het organiseren van een grondige pre-buy inspectie. Dit technisch onderzoek,
uitgevoerd door onafhankelijke experts, onthult potentiële onderhoudskosten
en kan aanzienlijke bedragen besparen. Lees onze
<a href="/kopen/privet-jet/inspectie/">complete inspectiegids</a>
voor een gedetailleerde checklist en wat u kunt verwachten.</p>
```

### Pattern 4: Location Mention → City Guide Link
```html
<p>Nederland biedt uitstekende faciliteiten voor privéjet-eigenaren, met
Amsterdam Schiphol als primaire hub. De luchthaven beschikt over meerdere
dedicated private jet terminals (FBO's) met eersteklas services. Voor een
volledig overzicht van faciliteiten, kosten en procedures, bekijk onze
<a href="/locaties/amsterdam/">Amsterdam privéjet-gids</a>.</p>
```

### Pattern 5: Aircraft Category → Specific Model Link
```html
<p>Voor Europese zakelijke missies zijn light jets vaak de meest economische
keuze, met aankoop prijzen tussen €1,5 en €5 miljoen. Populaire modellen in
deze categorie zijn de Cessna Citation CJ-serie en de
<a href="/vliegtuigen/embraer-phenom-300/">Embraer Phenom 300</a>,
die uitstekende prestaties combineert met lage operationele kosten.</p>
```

### Pattern 6: Problem → Solution Link
```html
<p>Veel eerste kopers onderschatten de complexiteit van vliegtuigfinanciering.
Nederlandse banken en gespecialiseerde luchtvaartleasemaatschappijen bieden
verschillende structuren, elk met eigen voor- en nadelen. Of u kiest voor
<a href="/kopen/privet-jet/leasing/">operating lease</a>,
<a href="/kopen/privet-jet/financiering/">kapitaallease, of een traditionele lening</a>,
de juiste financieringsstructuur kan honderdduizenden euro's besparen over de levensduur
van het vliegtuig.</p>
```

# Complete Sitemap - Phase 8

**Purpose:** 100+ page sitemap with bilingual URL structure, metadata, and phase assignments for Netherlands private jet market.

**URL Structure Convention:**
- **Primary Language:** Dutch (nl-NL) - default URLs
- **Secondary Language:** English (en-NL) - `/en/[path]` prefix for international buyers
- **Bilingual Implementation:** hreflang tags (Phase 9 technical SEO)

---

## URL Structure Patterns

### Pattern Definitions

| Content Type | URL Pattern | Example | Notes |
|--------------|-------------|---------|-------|
| Pillar Pages | `/kopen/[topic]` | `/kopen/privet-jet` | Primary buying journey pages |
| Cluster Pages | `/kopen/[topic]/[subtopic]` | `/kopen/privet-jet/financiering` | Supporting pillar content |
| Guide Pages | `/gids/[topic]` | `/gids/kosten` | Educational resources |
| City Pages | `/locaties/[city]` | `/locaties/amsterdam` | Location-specific content |
| Aircraft Pages | `/vliegtuigen/[model]` | `/vliegtuigen/gulfstream-g650` | Aircraft specifications |
| Business Pages | `/zakelijk/[topic]` | `/zakelijk/belasting` | B2B segment content |
| Blog/Supporting | `/blog/[topic]` | `/blog/verzekering-gids` | Long-tail supporting content |
| Tools/Calculators | `/tools/[tool-name]` | `/tools/kosten-calculator` | Interactive calculators |

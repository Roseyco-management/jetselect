<task type="auto">
  <name>Task 3: Implement canonical URL tags</name>
  <files>improved/index.html</files>
  <action>
    Add self-referencing canonical `<link>` tag in HTML `<head>` to prevent duplicate content issues:

    ```html
    <!-- Canonical URL -->
    <link rel="canonical" href="https://jetselect.nl/" />
    ```

    **Purpose:** Tells Google which version of a page is the "official" one when multiple URLs might show similar content (e.g., with/without trailing slash, with/without www, HTTP vs HTTPS).

    **Pattern for future pages (document in HTML comment):**
    ```html
    <!--
    Canonical Pattern for Pillar Pages:
    Each page should have self-referencing canonical pointing to its own URL.

    Example: /kopen/privet-jet page should have:
    <link rel="canonical" href="https://jetselect.nl/kopen/privet-jet" />

    IMPORTANT: Always use absolute URLs (https://jetselect.nl/...) not relative (/kopen/...).
    Always use Dutch version as canonical (not English /en/... version).

    (Apply to all 111 pages from Phase 8 SITEMAP.md during content production)
    -->
    ```

    **What to avoid:** Do NOT use relative URLs (/kopen/ instead of https://jetselect.nl/kopen/) - canonical must be absolute. Do NOT point to different page (canonical should be self-referencing unless intentionally consolidating duplicate content). Do NOT omit canonical on translated pages (both nl-NL and en-NL versions should have their own self-referencing canonical, not point to each other).
  </action>

  </verify>

# SERP Features Tracking - Phase 7

**Purpose:** Document SERP features for target keywords to identify optimization opportunities.

**Instructions for Data Entry (Plan 2):**
- For each primary keyword, manually search on Google.nl (incognito mode)
- Document which SERP features appear (Featured Snippets, PAA, Local Pack, etc.)
- Note which competitors own these features
- Identify opportunities where we can target featured snippets or PAA
- Look for patterns in what types of content trigger specific SERP features

**SERP Feature Types:**
- **Featured Snippet:** Direct answer box at top of results (paragraph, list, table)
- **People Also Ask (PAA):** Expandable question boxes
- **Local Pack:** Map with local business listings (3-pack)
- **Image Pack:** Horizontal row of images
- **Video Pack:** Video carousel
- **Knowledge Panel:** Entity information box (right side)
- **Site Links:** Additional links under main result
- **Reviews/Ratings:** Star ratings in search results
- **Related Searches:** "People also search for" and bottom suggestions
- **Top Stories:** News carousel
- **Shopping Results:** Product listings
